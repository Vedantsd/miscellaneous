Sentinel Search Algorithm

Understand how the optimization works.

How Sentinel Search Works

Step 1: Input the array and key from user

Step 2: Store the last element of array in a 'last' variable

Step 3: Put the key at the end of the array

Step 4: Now, it's guaranteed to get a result!

Step 5: Iterate through the loop to find the key

Step 6: If the result index is less than n, then the key exists in the array!

Step 7: If the result index is equal to n, then check equality of key and last

Step 8: If key == last, key exists, else, return "Key not found"

The core idea is to save an if-condition check inside the loop.

Program Implementation

def sentinel_search(arr, key):
    last_element = arr[-1] 
    
    arr[-1] = key 
    
    i = 0

    while arr[i] != key:
        i += 1
        
    arr[-1] = last_element
    
    if i < len(arr) - 1 or last_element == key:
        return i  # Key found at index i
    else:
        return -1
